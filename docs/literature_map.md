# Literature Map

## Open Robot Datasets

Open X-Embodiment, DROID, BridgeData, RoboNet, and related datasets already make broad robot-data scale and diversity central. This paper cannot claim that data coverage is new.

## Data Selection

Uncertainty sampling, embedding diversity, balanced selection, and failure prediction already select data under distribution shift.

## Mechanism Coverage

The defensible contribution is a mechanism-level coverage audit over contact, force/tactile, deformable, recovery, irreversibility, and long-horizon axes, plus local evidence that those axes predict held-out mechanism failures better than coarse coverage proxies.
